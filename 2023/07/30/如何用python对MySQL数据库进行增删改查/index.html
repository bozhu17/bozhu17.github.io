如何用python对MySQL数据库进行增删改查

2023/7/30

搭建环境

1.下载小皮
2.下载pycharm
3.如果文章图片没有显示，请自行挂代理

操作步骤

1.我们首先打开PHPstudy
2.启动MySQL
3.打开我们下载的pycharm，我们需要下载一个pymysql库，我们下载pymysql库有以下几个方法

方法一：

  • 我们通过命令提示符进行下载

输入 pip install pymysql

方法二：

  • 我们在下载的时候有可能下载不成功，这是因为这个库是在国外的一个平台，所有我们可以通过国内的镜像资源进行下载

输入 pip install pymysql -i 镜像网址: 或者 pip install -i 镜像网址 pymysql

  • 这里是国内一些python库的网址

清华大学：https://pypi.tuna.tsinghua.edu.cn/simple
豆瓣：https://pypi.douban.com/simple
阿里云：https://mirrors.aliyun.com/pypi/simple
中国科技大学：https://pypi.mirrors.ustc.edu.cn/simple

方法三：

  • 我们可以在pycharm中进行下载，我们点击左上角文件，选择设置,在弹出来的新页面中选择右侧的“+”，然后在新弹出的页面中搜索栏中搜索想要的库即可

  • 注意：在使用方法三的时候经常会显示安装错误，这可能是你是的pip版本和python版本不匹配导致的，我使用的python3.7，而我的pip版本是2.3版本的，所以我将pip2.3版本回退到2.0版本以后完美解决这个问题

python连接MySQL

一、调用pymysql库

import pymysql

二、连接MySQL数据库

a = pymysql.connect(host=”MySQL数据库的地址”,
port=3306,
user=’数据库用户名字’,
passwd=’数据库用户密码’,
charset=’utf8’
)

三、创建一个游标对象,游标对象用于执行MySQL语句并返回结果

  • 在Python中使用游标对象执行MySQL语句需要使用execute()方法。execute()方法需要传入要执行的MySQL语句。

cur = a.cursor()

四、查询MySQL数据

  • 在Python中查询MySQL数据需要使用SELECT语句。可以使用execute()方法执行SELECT语句，并使用fetchall()方法获取查询结果。

cur.execute(‘show TABLES’)
chakan2 = cur.execute(‘select * from 表名’)

五、插入数据到MySQL数据库

  • 在Python中插入数据到MySQL数据库需要使用INSERT语句。可以使用execute()方法执行INSERT语句，并使用commit()方法将更改提交到数据库。

h = ‘INSERT INTO 表名(你要插入内容的表头) VALUE(%s,%s,%s,%s)’
value = (写入的数据)
cur.execute(h,value)

六、更新MySQL数据

  • 在Python中更新MySQL数据需要使用UPDATE语句。可以使用execute()方法执行UPDATE语句，并使用commit()方法将更改提交到数据库。

xiugai = “update 表名 SET 修改的表头=%s WHERE 修改的表头=%s”
value = (‘修改值’,’被修改值’)
cur.execute(xiugai,value)

七、删除MySQL数据

  • 在Python中删除MySQL数据需要使用DELETE语句。可以使用execute()方法执行DELETE语句，并使用commit()方法将更改提交到数据库。

shanchu = “DELETE FROM 表名 WHERE NAME=%s”
value = (‘修改值’)
cur.execute(shanchu,value)
