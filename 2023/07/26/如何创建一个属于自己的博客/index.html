如何创建一个属于自己的博客

2023/7/26 教程

准备前提

  1. 提前安装好(node.js)[https://nodejs.org/en] 左边的那个
  2. 提前安装好(GIt)[https://git-scm.com/downloads]
  3. 提前注册好一个github账号
  4. 如果文章图片没有显示，请自行挂代理

1.查看环境

1.输入上面两个网址下载安装完成以后，键盘上按下Win+R键
2.在弹出的对话框中输入CMD
3.在命令提示符中输入npm -version，查看npm是否安装成功

4.如果显示nmp不是内部或外部命令，表示需要我们手动搭建环境
5.双击此电脑，在空白处右击选择属性
6.选择高级系统设置，在对话框下方选择环境变量
7.在新出现的对话框下方系统变量中找到path，双击进入
8.点击新建，输入nodejs软件的安装位置，指向node.exe所在的文件夹

2.操作步骤

1.在本地磁盘中创建一个文件夹，路径不要有中文
2.选中刚刚创建的目录，右键在菜单栏中选择Git Bash Here
3.使用npm命令安装Hexo，输入：

npm install hexo-cli -g
hexo init blog
cd blog

构建好的效果：

4.进入blog目录后，我们需要按顺序输入以下三条命令用来初始化我们的博客：

hexo new test_my_site
hexo g
hexo s

输入完成后显示如下命令后即可通过127.0.0.1:4000访问blog了

5.配置ssh

cd ~/.ssh

  • 如果出现提示：No such file or directory 说明你是第一次使用git

ssh-keygen -t rsa -C”邮箱地址”

  • 连续按下三次回车，会生成一个文件在用户目录下，打开用户目录，找到.ssh\id_rsa.pub文件，记事本打开复制其中内容，前往网站打开GitHub主页，点击个人设置——>SSH and GPG keys ——> New SSH key：

  • 将刚刚复制的内容写入key中，输入下面的命令查看SSH是否连接成功

ssh -T git@github.com

  • 成功就会显示如下命令，输入yes即可

  • 然后在上方的地址栏中输入cmd，回车会弹出当前的命令提示符，输入如下代码：

npm install hexo-deployer-git -save

  • 注意：如果显示权限不够，可以从开始菜单搜索命令提示符，右键以管理员模式运行，输入：

cd D:\hexo\blog# 我存放的位置是D盘，你们只需要找到自己的blog文件位置即可

  • 回到Git Bash Here界面继续配置

git config –global user.name “luozhixiaowo”// 你的github用户名，非昵称
git config –global user.email “xxx@qq.com“// 填写你的github注册邮箱

6.上传代码

  • 在github上新建一个存储库，名字格式为 github用户名.github.io

  • 编辑_config.yml文件，再最后改成这样

  • 注意：新创建的存储库一定要和用户名一致，不一致可能会出现错误，更改文件一定要保持格式，缩进不对，也会导致上传不成功

  • 上传文件

hexo d
每次上传文件后，需要在github上等待几分钟
